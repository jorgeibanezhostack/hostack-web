Database Setup

Open this file locally in your browser. It creates the admin login user automatically. For the database schema, you can either paste the SQL into Supabase SQL Editor, or provide a Personal Access Token for full automation.

Step 1 — Credentials
Without a PAT: schema SQL will be shown for manual copy-paste into Supabase SQL Editor.
With a PAT: the full schema runs automatically. Get one at supabase.com → Account → Access Tokens → Generate new token.
